## 📌 Project Overview

ImagineAI is a responsive AI-powered image generation web application that converts text prompts into AI-generated images. Users can enter creative prompts, choose different art styles, generate images, download results, save favorites, and maintain prompt history.

## 🚀 Features

* Generate AI images from text prompts
* Multiple art style options

  * Realistic
  * Anime
  * Fantasy
  * Cartoon
  * Digital Art
  * 3D Render
  * Cyberpunk
* Download generated images
* Favorites section
* Prompt history tracking
* Responsive design for mobile and desktop
* Dark/Light theme toggle
* Full-screen image preview
* Modern and interactive UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Pollinations AI Image API
* Local Storage
